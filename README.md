# Brain-Tumor-Detection-System

## Problem
Early detection of brain tumors is critical for effective treatment and improved patient outcomes. Manual analysis of MRI scans can be time-consuming and requires expertise, making automation detection systems valuable in medical diagnostics.

## Approach
Developed a deep learning-based image classification model using a Convolutional Neural Network (CNN) to detect the presence of brain tumors from MRI images. The model processes medical image data and classifies it into tumor and non-tumor categories.

## Technologies Used 
- Python
- TensorFlow / Keras
- Numpy
- Matplotlib

## Results
- Successfully trained a CNN model for tumor classification
- Achieved prediction capability on MRI image data
- The model can distinguish between tumor and non-tumor cases

## Future Improvements
- Improve model accuracy with larger and more diverse datasets
- Implement real-time prediction interface
- Deploy as a web-based diagnostic support tool
